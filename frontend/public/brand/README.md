# Assets de marca

Añade aquí tus archivos SVG:

- **`logo.svg`** — Logo / icono de la app (se usará en la cabecera, favicon, etc.).
- **`pattern.svg`** — Patrón de fondo para el diseño (se usará como textura o background).

Ambos deben ser archivos `.svg`. Una vez añadidos, se podrán referenciar en la app como `/brand/logo.svg` y `/brand/pattern.svg`.
